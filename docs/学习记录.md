## 4.字符串拼接

传统方式：

```python
print("你好：" + name)
```

数字不能直接拼接：

错误：

```python
print("年龄：" + age)
```

正确：

```python
print("年龄：" + str(age))
```

---

## 5.f-string（推荐）

```python
print(f"你好，{name}")
print(f"年龄：{age}")
```

以后项目开发推荐使用这种方式。

# 三、今日易错点

## ① Python 文件找不到

错误：

```text
python hello.py
```

提示：

No such file or directory

原因：

终端目录错误。

正确：

```powershell
cd python-learning\lesson01
python hello.py
```

---

## ② 字符串不能和整数相加

错误：

```python
print("总分：" + 270)
```

报错：

```text
TypeError
```

正确：

```python
print("总分：" + str(270))
```

或者

```python
print(f"总分：{270}")
```

推荐第二种。

---

## ③ input()

错误：

```python
age = input()
print(age + 1)
```

原因：

input() 返回字符串。

正确：

```python
age = int(input())
print(age + 1)
```

---

## ④ f-string

错误：

```python
print(f"age")
```

输出：

```text
age
```

正确：

```python
print(f"{age}")
```

变量必须写在 `{}` 中。

# 七、面试角

### Q1：为什么 input() 返回的是字符串？

答：

因为用户输入的内容本质上是文本，所以 Python 默认返回 str。

需要计算时再转换：

```python
age = int(input())
```

---

### Q2：为什么推荐 f-string？

优点：

- 可读性高
- 性能更好
- 不需要频繁使用 `+`
- Python 官方推荐

例如：

```python
print(f"姓名：{name}，年龄：{age}")
```

---

### Q3：type() 有什么作用？

查看变量的数据类型：

```python
print(type(age))
```
