# 一、while 循环

语法：

```python
while 条件:
    循环体
```

例如：

```python
count = 1

while count <= 5:
    print(count)
    count += 1
```

---

# 二、while 与 for 的区别

| while | for |
|--------|-----|
| 不知道循环多少次 | 已知循环次数 |
| 根据条件结束 | 根据序列结束 |
| 登录、菜单、游戏 | 遍历列表、字符串、range |

---

# 三、while True

语法：

```python
while True:
```

特点：

一直循环。

通常配合：

```python
break
```

一起使用。

应用场景：

- AI 对话
- 游戏
- 登录系统
- 控制台菜单

---

# 四、break

作用：

结束整个循环。

示例：

```python
count = 1

while True:
    print(count)

    if count == 3:
        break

    count += 1
```

输出：

```
1
2
3
```

---

# 五、continue

作用：

结束本次循环。

继续下一次循环。

示例：

```python
count = 0

while count < 5:
    count += 1

    if count == 3:
        continue

    print(count)
```

输出：

```
1
2
4
5
```

---

# 六、break 与 continue 对比

| break | continue |
|---------|-----------|
| 结束整个循环 | 跳过当前循环 |
| 后续循环不会执行 | 下一轮循环继续执行 |

# 七、常见错误

## 1、忘记修改计数器

错误：

```python
while count <= 5:
    print(count)
```

结果：

死循环。

---

## 2、break 放置位置错误

break 前后的代码执行顺序不同。

应该认真分析程序执行流程。

---

## 3、continue 后代码不会执行

例如：

```python
continue

print("Hello")
```

Hello 永远不会输出。
