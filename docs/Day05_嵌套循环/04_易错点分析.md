# Day05｜易错点分析

## 1、REPL 中运行 Python 文件

错误：

```text
>>>
python demo.py
```

报错：

```text
SyntaxError
```

原因：

REPL 只能执行 Python 代码。

不能执行命令。

正确方式：

在终端运行：

```bash
python demo.py
```

或：

> 在专用终端运行 Python 文件。

---

## 2、Git push 被拒绝

原因：

GitHub 已存在 README。

本地仓库没有。

解决：

```bash
git pull origin main --allow-unrelated-histories

git push
```

---

## 3、range()

容易忘记：

stop 不包含。

例如：

```python
range(1,6)
```

实际结果：

```
1 2 3 4 5
```

---

## 4、break 与 continue

break：

结束整个循环。

continue：

跳过当前循环。

不要混淆。

---

## 5、print()

写图形时容易忘记：

```python
end=""
```

导致输出结果全部换行。
