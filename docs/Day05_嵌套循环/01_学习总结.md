# Day05｜学习总结

## 一、今日学习内容

今天主要学习了 Python 的循环语句。

### 1、while 循环

掌握了 while 的基本语法：

```python
count = 1

while count <= 5:
    print(count)
    count += 1
```

理解了：

- 初始化变量
- 判断条件
- 更新变量

缺一不可，否则容易进入死循环。

---

### 2、break

学习了提前结束循环。

例如：

```python
count = 1

while True:
    print(count)

    if count >= 3:
        break

    count += 1
```

break 用于立即结束整个循环。

---

### 3、continue

学习了跳过本次循环。

例如：

```python
count = 0

while count < 5:
    count += 1

    if count == 3:
        continue

    print(count)
```

continue 不会结束循环，只会跳过当前这一次。

---

### 4、for 循环

学习了：

```python
for i in range(1,6):
    print(i)
```

理解了：

for 用来遍历数据。

---

### 5、range()

掌握三种写法：

```python
range(5)

range(1,6)

range(10,1,-2)
```

并理解：

- start
- stop
- step

---

### 6、嵌套循环

完成：

```python
for i in range(1,3):
    print(f"第{i}天")

    for j in range(1,4):
        print(f"第{j}节课")
```

理解：

外层循环控制次数。

内层循环控制每次内部执行内容。

---

### 7、九九乘法表

独立完成：

```python
for i in range(1,10):
    for j in range(1,i+1):
        print(f"{j}*{i}={i*j}",end=" ")
    print()
```
