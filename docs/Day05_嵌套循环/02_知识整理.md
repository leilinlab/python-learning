# Day05｜知识整理

## while

```python
while 条件:
    循环体
```

特点：

只要条件成立，就一直执行。

---

## break

作用：

结束整个循环。

---

## continue

作用：

跳过当前循环。

继续下一次循环。

---

## for

语法：

```python
for i in range():
```

适用于遍历数据。

---

## range()

### 第一种

```python
range(5)
```

结果：

```
0
1
2
3
4
```

---

### 第二种

```python
range(1,6)
```

结果：

```
1
2
3
4
5
```

---

### 第三种

```python
range(10,1,-2)
```

结果：

```
10
8
6
4
2
```

---

## len(range())

```python
r = range(1,10,2)

print(len(r))
```

结果：

```
5
```

---

## 嵌套循环

格式：

```python
for i in range():
    for j in range():
```

外层循环一次，

内层循环完整执行一次。

---

## print()

取消换行：

```python
print("*",end="")
```

主动换行：

```python
print()
```
